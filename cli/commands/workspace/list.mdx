## Overview

List all your workspaces with their status, creation dates, and descriptions. Shows which workspace is currently active.

## Syntax

```bash
ntcli workspace list [options]
```

## Aliases

```bash
ntcli workspace ls    # Short alias
ntcli ws list         # Command group alias
ntcli ws ls           # Combined aliases
```

## Options

<ResponseField name="-v, --verbose" type="boolean">
  Show detailed information including workspace IDs and resource usage
</ResponseField>
