## What is NimbleSearch?

NimbleSearch is a web search service powered by Brave Search. Use it to:

- Research topics and gather information
- Find current news and updates
- Look up company information
- Verify facts and data

## Example Usage

### Research a Topic

```
Search for the latest news about artificial intelligence regulations in the EU
```

### Company Research

```
Search for information about Acme Corp's recent product launches
```

### Fact Verification

```
Search to verify the current population of Tokyo, Japan
```

### Competitive Analysis

```
Search for reviews and comparisons of project management tools
```

## Limitations

- **Rate limits**: Reasonable search limits apply
- **No deep crawling**: Returns search results, not full page content
- **General search**: Not optimized for specific domains

## When to Use NimbleSearch vs Tavily

| Use Case | NimbleSearch | Tavily |
|----------|--------------|--------|
| General web search | Yes | Yes |
| AI-optimized results | No | Yes |
| Content extraction | No | Yes |
| Site crawling | No | Yes |
| No setup required | Yes | No (API key required) |
| Free | Yes | Paid tier |

<Tip>
Use **NimbleSearch** for basic research and fact-finding. Use **Tavily** when you need AI-optimized results, content extraction, or site crawling capabilities.
</Tip>
