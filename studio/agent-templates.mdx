## Personalizing Your Agents

Each agent can be customized during onboarding or anytime:

<Steps>
  <Step title="Choose Topics">
    Tell the agent what you're interested in tracking (AI, startups, finance, etc.)
  </Step>
  <Step title="Set Location">
    For weather and local context (Casey and Alex use this)
  </Step>
  <Step title="Configure Schedule">
    Set when you want email briefings delivered
  </Step>
  <Step title="Start Chatting">
    Mention the agent by name to interact: `@riley`, `@alex`, `@casey`
  </Step>
</Steps>

## Adding More Agents

Beyond the pre-built templates, you can create custom agents tailored to your specific needs:

1. Go to **Agents** in the sidebar
2. Click **Create Agent**
3. Configure the agent's personality and instructions
4. Connect relevant playbooks
5. Set up schedules if needed
