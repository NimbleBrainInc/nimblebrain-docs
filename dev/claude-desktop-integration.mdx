## Overview

Connect your deployed MCP servers to Claude Desktop for seamless AI-powered workflows with real-time data access.

## Integration Architecture

```mermaid
graph LR
    Claude[Claude Desktop] --> Bridge[MCP HTTP Bridge]
    Bridge --> API[NimbleTools API]
    API --> Server[MCP Server]
```

## Setup Process

<Steps>
  <Step title="Deploy Server">
    ```bash
    ntcli server deploy nationalparks-mcp
    ```
  </Step>
  <Step title="Set Secret">
    ```bash 
    ntcli secrets set NPS_API_KEY [YOUR_API_KEY]
    ```
  </Step>

  <Step title="Generate Config">
    ```bash  
    ntcli server claude-config nationalparks-mcp
    ```

## Configuration Locations

<Tabs>
  <Tab title="macOS">
    `~/Library/Application Support/Claude/claude_desktop_config.json`
  </Tab>
  <Tab title="Linux">`~/.config/claude/claude_desktop_config.json`</Tab>
  <Tab title="Windows">`%APPDATA%\Claude\claude_desktop_config.json`</Tab>
</Tabs>
