# Integrations

Connect your NimbleBrain workspace to popular AI development tools and platforms. NimbleBrain provides MCP server integrations that work seamlessly with Claude Desktop, Claude Code, Cursor, and other MCP-compatible clients.

## Integration Overview

All integrations follow the same basic pattern:

<Steps>
  <Step title="Deploy MCP Servers">
    Install the MCP servers you need in your NimbleBrain workspace
  </Step>
  <Step title="Generate Configuration">
    Get the connection configuration from the NimbleBrain Studio
  </Step>
  <Step title="Configure Client">
    Add the configuration to your client application (Claude Desktop, Claude Code, etc.)
  </Step>
  <Step title="Start Using">
    Access your MCP tools directly from the integrated client
  </Step>
</Steps>
