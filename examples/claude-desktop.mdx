## Overview

Learn how to integrate deployed MCP servers with Claude Desktop for seamless AI-powered workflows.

## Prerequisites

- ntcli installed and authenticated
- At least one MCP server deployed
- Claude Desktop installed

## Step-by-Step Integration

<Steps>
  <Step title="Deploy MCP Server">
    ```bash
    ntcli server deploy nationalparks-mcp
    ```
  </Step>
  <Step title="Generate Configuration">
    ```bash
    ntcli server claude-config nationalparks-mcp
    ```
  </Step>
  <Step title="Add to Claude Desktop">
    Copy the output to your claude_desktop_config.json file
  </Step>
  <Step title="Restart Claude Desktop">
    Restart Claude Desktop to load the new configuration
  </Step>
</Steps>

## Configuration Files

<Tabs>
  <Tab title="macOS">
    `~/Library/Application Support/Claude/claude_desktop_config.json`
  </Tab>
  <Tab title="Linux">
    `~/.config/claude/claude_desktop_config.json`  
  </Tab>
  <Tab title="Windows">
    `%APPDATA%\Claude\claude_desktop_config.json`
  </Tab>
</Tabs>
